Cech Cohomology

One formal project I have been doing over the last few years is an expository project on Cech Cohomology. It began as an introductory project on Riemann surfaces at Canada/USA Mathcamp, and gently introduced sheaves during a Directed Reading Project. The project culminated in the second half of this expository paper. While the project was necessarily very deep, I hope to do more interesting things with cohomology with category theory that I develop from Modern Algebra this year.

LaTeX Note taking

A more informal recent project I have been undertaking is in how I write up my homework on LaTeX. Using vim snippets, Gilles Castel introduced an extremely efficient way to write down clean, elegant math on LaTeX that is fast enough to be suitable for class notes. Of course, I had to introduce some shortcuts exclusive to my classes (like epsilon for real analysis or mathcal for category theory). I definitely think that this way of writing LaTeX will come in handy for writing accessible presentations in the future.
